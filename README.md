# ui-ssot

UI sample projects (templates / reference implementations). Each sample is self-contained.

## Samples

- `hp-sample/` — Next.js (App Router) + Tailwind corporate website template (13 locales)
- `ml-sample/` — Next.js (App Router) music library UI sample (template-ready)

## Quick start

```bash
cd hp-sample
npm install
cp .env.example .env.local
npm run dev
```

```bash
cd ml-sample
npm install
npm run dev
```

Verification:

```bash
cd hp-sample
npm run verify
```

```bash
cd ml-sample
npm run verify
```
