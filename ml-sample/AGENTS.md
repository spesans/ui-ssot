# AGENTS.md

## Project overview

`ml-sample/` is a music library UI sample (template-ready) built with Next.js (App Router).

## Setup commands

- Install dependencies: `npm install`
- Start dev server: `npm run dev`

## Build & verification

- Verification gate (MUST): `npm run verify`

## Code style

- Lint config: `eslint.config.mjs` (ESLint v9 flat config + typed linting)
- Format config: `prettier.config.cjs`
